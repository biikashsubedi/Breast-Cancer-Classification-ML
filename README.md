# 🔬 Breast Cancer Classification Predictor

> A machine learning project that predicts whether a breast cancer tumor is malignant or benign based on diagnostic measurements.

## 🚀 Overview
Early detection of breast cancer greatly improves survival rates. This project applies supervised machine learning techniques to diagnostic data to classify tumors. By comparing different algorithms, the goal is to build a model that maximizes prediction accuracy and minimizes false negatives (which are critical in medical diagnoses).

## ✨ Features & Methodology
* **Exploratory Data Analysis (EDA):** Visualized feature distributions and correlations using Seaborn and Matplotlib.
* **Logistic Regression:** Implemented as the baseline binary classification model.
* **K-Nearest Neighbors (KNN):** Implemented to compare performance against the linear baseline.
* **Model Evaluation:** Evaluated models using Accuracy, Precision, Recall, and ROC-AUC scores.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** Scikit-Learn (`LogisticRegression`, `KNeighborsClassifier`)
* **Data Manipulation:** Pandas
* **Data Visualization:** Seaborn, Matplotlib
